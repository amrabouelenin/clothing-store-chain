Clothing Shop - Branch Office

This system is utilizing Json API approach and expose web services for different types of client in order to interact with the backend database, currently the branch office only handles products and transaction management. and it was meant to prove how both backend and clients will be able to work togeather. Below are the API urls that has been used for that purpose:

  • http://127.0.0.1:8000/api/add/product/
  • http://127.0.0.1:8000/api/list/products/
  • http://127.0.0.1:8000/api/add/transaction/.
  • http://127.0.0.1:8000/api/list/transactions/.
